# 📚 University & Computer Science Hub

Bienvenido a mi repositorio central. Este espacio está diseñado para organizar de manera estructurada todo el material académico de mi formación universitaria, guías de ejercicios y proyectos personales de Computer Science.

## 🗂️ Estructura del Repositorio

El repositorio se divide en grandes bloques para facilitar la navegación:

* **!!Neo_Today**: Carpeta de trabajo activo donde se encuentran los proyectos y materias del semestre actual.
* **!!Neo_Old**: Archivo histórico de materias y proyectos finalizados.
* **Computer Science Projects**: Desarrollos de software, algoritmos y configuraciones técnicas (como optimización de hardware y GPU).

## 🎓 Contenido Académico

Aquí gestiono material de diversas áreas críticas, incluyendo:
* **Teoría de la Computación**: Conceptos fundamentales y algoritmos.

## 🛠️ Tecnologías Utilizadas

* **LaTeX**: Para la creación de guías de estudio profesionales y documentos académicos con alta calidad tipográfica.
* **Git & GitHub**: Control de versiones para asegurar que mi progreso académico esté respaldado y organizado.
* **VS Code**: Mi entorno de desarrollo principal para código y documentación.
